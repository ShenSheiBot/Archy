## 🚀 强烈推荐 (High Priority)

### 4. 快速切换置顶/取消置顶 ⭐⭐⭐⭐
**场景**：有时候想临时让窗口不置顶，比如要点击下面的窗口
- [ ] 添加快捷键切换 always-on-top 状态
- [ ] 建议快捷键：Cmd+Shift+P (Pin/Unpin)
- [ ] 在设置面板显示当前置顶状态
- [ ] 可选：添加视觉指示器（图标或边框变化）

**实现难度**：低

---

### 5. 鼠标穿透模式改进 ⭐⭐⭐
**当前状态**：已有 Detached Mode (Cmd+Shift+D)，但需要点击才能恢复
- [ ] 改进：添加快捷键直接退出穿透模式
- [ ] 或：鼠标移到窗口边缘时自动恢复
- [ ] 添加视觉指示器显示当前在穿透模式
- [ ] 记住穿透模式状态（session restore）

**实现难度**：低-中等

---

### 6. 网页截图 ⭐⭐⭐
**场景**：浮动窗口用来参考，截图很有用
- [ ] 可见区域截图（不需要整页截图）
- [ ] 快捷键：Cmd+Shift+S
- [ ] 保存到剪贴板 + 可选保存到文件
- [ ] 截图时显示提示（flash 效果）

**实现难度**：低

---

## 💡 Nice to Have (Medium Priority)

### 7. 缩放预设/快速切换 ⭐⭐
**场景**：小窗口看文档时需要快速调整缩放
- [ ] 预设缩放级别：50%, 75%, 100%, 125%, 150%
- [ ] 快捷键循环切换（Cmd+= / Cmd+-）
- [ ] 或：快捷键直接跳到 100% (Cmd+0)
- [ ] 在顶栏显示当前缩放级别

**实现难度**：低

---

### 8. 快速访问列表 ⭐⭐
**场景**：几个常用网址的快速启动（比书签更轻量）
- [ ] 预设常用网站列表（可配置）
  - YouTube
  - Bilibili
  - Twitter
  - Reddit
  - GitHub
  - Stack Overflow
- [ ] 通过快捷键或下拉菜单快速打开
- [ ] 用户可自定义列表

**实现难度**：低-中等

---

### 9. Compact 模式 ⭐⭐
**场景**：小窗口模式下节省空间
- [ ] 更小的顶栏
- [ ] 更少的边距和间距
- [ ] 可切换：Normal / Compact 模式
- [ ] 快捷键：Cmd+Shift+C

**实现难度**：中等

---

### 10. 启动选项配置 ⭐⭐
- [ ] 启动时行为可配置：
  - 空白页
  - 恢复上次的页面（当前行为）
  - 指定 URL
- [ ] 启动时窗口状态：
  - 显示/隐藏
  - 记住上次状态
- [ ] 添加到设置面板

**实现难度**：低
